1.Apresente 5 sistemas operacionais suportados pelo Raspberry Pi, e algumas de suas características.
Raspbian - é uma versão do Debian e é bem completo, tem softwares de escritório e permite navegar na internet, possui
ferramentas de configuração do computador e de desenvolvimento.
Ubuntu MATE - possui aplicativos embarcados do Ubuntu, é possível baixar novas ferramentas e é um desktop mais
tradicional.
OSMC - é um sistema operacional para tornar o Raspberry em um media center, permite a reprodução de aúdio e vídeo com
qualidade.
Racalbox - transforma o Raspberry em uma central de emulação de jogos, possui configurações amgáveis e possui suporte para 
consoles como Ataria, Super Nintendo, Playstation e etc.
Pidora - Tem configurações parecidas com o Ubuntu Mate, o nome vem da junção de Pi com Fedora, uma distribuição Linux.

2.Apresente as formas de instalação de sistemas operacionais para o Raspberry Pi.
Pode-se instalar através de um SD card onde é colocado um sistema operecional ou um instalador.

3.Apresente 3 formas de desenvolvimento de software para o Raspberry Pi.
Compilação cruzada no computador anfitrião e enviar executáveis pela conexão SSH, compilação direta no Raspberry 
por meio de conexão pelo SSH e compilação cruzada e colocar executáveis diretamente no sistema de arquivos.

4.Apresente 3 formas de acesso remoto ao Raspberry Pi.
SSH via Wi-Fi, SSH via rede pelo cabo Ethernet, terminal UART.

5.Apresente as formas possíveis de compilação de código em C para o Raspberry Pi.
Compilação diretamente no Raspberry através do terminal usando o GCC, compilação em outro computador através de uma 
toolchain que permita compilação cruzada.

6.Apresente as formas possíveis de compilação de código em Python para o Raspberry Pi.
Um código em Python necessita do interpretador Python para ser execultado no terminal.
